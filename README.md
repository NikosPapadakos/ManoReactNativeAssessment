## How to Run

1. Install the necessary packages:
   ```bash
   yarn
   ```

2. Install CocoaPods dependencies:
   ```bash
   npx pod install
   ```

3. Run the project on iOS or Android:
   ```bash
   yarn run ios
   ```

   or

   ```bash
   yarn run android
   ```
